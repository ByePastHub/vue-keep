# vue-keep-router-view

## 介绍

>`vue-keep-router-view`可以帮助你实现前进刷新，返回时之前的页面还处于激活的状态，可以像`app`一样的体验。哪里需要缓存，把`router-view`组件使用`keep-router-view`来替换。

vue2.x
```

```

```
npm install vue-keep-router-view
```
